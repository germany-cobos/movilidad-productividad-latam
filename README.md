# movilidad-productividad-latam
Data analysis project exploring how urban mobility impacts economic productivity in Latin American cities using Python, TomTom Traffic Index and OECD data.
# Movilidad urbana y productividad económica en LATAM

## Descripción
Este proyecto analiza la relación entre la congestión urbana y la productividad económica en ciudades de América Latina.

## Objetivo
Identificar ciudades con:
- Alta congestión y baja productividad
- Mejores indicadores combinados

## Datos utilizados
- TomTom Traffic Index
- OECD Cities

## Herramientas
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Proceso
- Limpieza y unión de datasets
- Análisis exploratorio (EDA)
- Visualización de datos

## Resultados
Se identificaron patrones entre congestión y desempeño económico, permitiendo detectar oportunidades de mejora en infraestructura urbana.
